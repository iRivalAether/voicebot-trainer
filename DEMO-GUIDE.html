🎤 Voicebot Entrenador Personal - Guía de Demo

Esta guía te ayudará a probar todas las funcionalidades del Voicebot Personal Trainer.

🚀 Pasos para Inicializar

# 1. Navegar al proyecto
cd C:\xampp\htdocs\voicebot-trainer

# 2. Instalar dependencias
npm install

# 3. Configurar environment (opcional)
cp .env.example .env

# 4. Iniciar servidor de desarrollo
npm run dev

# 5. Abrir http://localhost:3000

🎯 Casos de Prueba

🏋️ Ejercicios

Test 1: Presiona 🎤 y di: "Dame ejercicios para brazos"
Esperado: Lista de ejercicios de brazos + explicación hablada
Test 2: Escribe: "rutina de piernas para principiantes"
Esperado: Ejercicios de piernas adaptados
Test 3: Voz: "ejercicios de pecho sin equipo"
Esperado: Ejercicios de cuerpo libre

🥗 Nutrición

Test 4: Voz: "qué puedo desayunar saludable"
Esperado: Opciones de desayuno + consejos nutricionales
Test 5: Escribe: "dame ideas para una cena ligera"
Esperado: Sugerencias de cena + tips
Test 6: Voz: "snacks saludables después del gym"
Esperado: Snacks post-workout

💪 Motivación

Test 7: Voz: "dame motivación para entrenar"
Esperado: Mensaje motivacional + tono encouraging
Test 8: Escribe: "necesito ánimos"
Esperado: Consejo motivacional personalizado

🔧 Funcionalidades Técnicas

Test 9: Botón 🔇 al lado de respuestas del bot
Esperado: Reproduce la respuesta nuevamente
Test 10: Verificar logs en consola del navegador (F12)
Esperado: Ver logs de interacciones y APIs
Test 11: Probar sin conexión de APIs
Esperado: Funciona con datos de fallback

🔍 Verificaciones Técnicas

Consola del Navegador (F12)

• 🤖 AI Analysis: Muestra el análisis de intención de Gemini
• 🎤 Voice recognition started/ended: Estado del micrófono
• 🗣️ Speech recognized: Texto detectado por voz
• 📊 User Interaction Log: Registro de conversaciones
• ✅ Interaction logged: Confirmación de logging

Network Tab (APIs)

• Gemini API: Calls to generativelanguage.googleapis.com
• ExerciseDB: Calls to exercisedb.p.rapidapi.com (si está configurada)
• Advice Slip: Calls to api.adviceslip.com
• Webhook: POST a tu URL de webhook (si está configurada)

📱 Compatibilidad

✅ Navegadores Recomendados: Chrome 80+, Edge 80+, Safari 14+
⚠️ Limitaciones: Firefox (limitaciones en Web Speech API)
📱 Mobile: Funciona en móviles, pero mejor experiencia en desktop
🔐 HTTPS: Requerido para funcionalidades de voz en producción

🚀 Despliegue

Build de Producción

npm run build
npm run preview # Para probar el build localmente

Deploy a Vercel

# Instalar Vercel CLI
npm i -g vercel

# Deploy
vercel --prod

# Configurar environment variables en dashboard
